# Tomato Disease Classifier 🍅🤖

AI-powered Telegram bot for detecting tomato plant diseases using deep learning.

## 🚀 Features
- 📸 Disease detection from photos
- 💬 Telegram bot interface  
- ⚡ Real-time predictions
- 🎯 10 disease types classification

## 🛠 Tech Stack
- **Framework:** PyTorch
- **Language:** Python 3.10
- **Bot:** Telegram Bot API
- **Model:** ResNet-based CNN
