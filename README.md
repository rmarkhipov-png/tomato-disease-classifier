# Tomato Disease Classifier 🍅🤖

AI-powered Telegram bot for detecting tomato plant diseases using deep learning.

## 🚀 Features
- 📸 Disease detection from photos
- 💬 Telegram bot interface  
- ⚡ Real-time predictions
- 🎯 10 disease types classification

## 🛠 Tech Stack
- **Framework:** PyTorch
- **Language:** Python 3.10
- **Bot:** Telegram Bot API
- **Model:** ResNet-based CNN

## 📦 Installation

\\\ash
# Clone repository
git clone https://github.com/rmarkhipov-png/tomato-disease-classifier.git
cd tomato-disease-classifier

# Install dependencies
pip install -r requirements.txt

# Set up Telegram bot token
export TELEGRAM_BOT_TOKEN='your_token_here'
\\\

## 🚀 Usage

\\\ash
# Run the bot
python src/bot.py

# Or train new model
python scripts/train.py
\\\
